Qu'est-ce qu'un test statique ?

Les tests statiques consistent à examiner le code ou la documentation sans exécuter le programme. Ils permettent de détecter des erreurs avant même que le logiciel ne soit testé dynamiquement.

  • Exemple : Vérification de syntaxe dans un code.
  • Exemple : Relecture de documents techniques.
