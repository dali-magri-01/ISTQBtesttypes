Qu'est-ce qu'un test non fonctionnel ?

Les tests non fonctionnels évaluent comment le logiciel fonctionne plutôt que ce qu'il fait. Ces tests mesurent des aspects tels que la performance, la compatibilité, l'utilisabilité, la sécurité et plus encore.

  • Exemple : Tester la rapidité de chargement d'une page web.
  • Exemple : Vérifier si une application fonctionne correctement sur différents navigateurs.
