Qu'est-ce qu'un test dynamique ?

Les tests dynamiques sont effectués en exécutant le programme pour valider son comportement. Contrairement aux tests statiques, ils vérifient si le code fonctionne correctement lors de son exécution.

  • Exemple : Tester si une fonction retourne la bonne valeur avec une entrée donnée.
  • Exemple : Vérifier qu'une application ne plante pas après un clic sur un bouton.
